## Short Notes

- [Introduction to Algorithms (CLRS)](Thomas%20H.%20Cormen,%20Charles%20E.%20Leiserson,%20Ronald%20L.%20Rivest,%20Clifford%20Stein%20Introduction%20to%20Algorithms,%20Third%20Edition%20%202009.pdf): Core algorithms and correctness foundations.
- [Programming Challenges](Programming_Challenges.pdf): Classic ACM/ICPC problem-solving practice.
- [Competitive Programming 3](Competitive%20Programming%203.pdf): Contest patterns and coding templates.
- [CP2](cp2.pdf): Compact competitive programming fundamentals.
- [CP4 Book 1](imp-cp4-1.pdf): Structured intermediate contest training.
- [CP4 Book (General)](imp-cp_book.pdf): Broad CP curriculum and practice index.
- [ICPC CP4 Book 2](imp-icpc-cp4book2.pdf): Advanced ICPC-level scenarios.
- [Looking for a Challenge (ICPC Questions)](Looking%20for%20a%20Challenge-icpc-programming-question.pdf): Harder, creative challenge sets.
- [Concrete Mathematics](Concrete%20Mathematics%20A%20Foundation%20for%20Computer%20Science~tqw~_darksiderg.pdf): Discrete math for proofs and analysis.
