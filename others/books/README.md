# Other Books

This folder contains a small set of books with short notes so you can quickly see what each one is about.

## Book Notes

- [The Accidental CTO](The%20Accidental%20CTO%20Book.pdf): Practical guidance on technical leadership, decision-making, and growing into a CTO role.
- [Data Structures and Algorithms in Java](data_structures_and_algorithms_in_java_4th_edition.pdf): Core Java-based DSA foundations with implementation-focused examples.
- [Writing an Interpreter in Go](writing_an_interpreter_in_go.pdf): Step-by-step language implementation, parser design, and interpreter construction.
