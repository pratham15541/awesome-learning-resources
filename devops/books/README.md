# DevOps Books

This folder contains the local reading list for DevOps and production operations.

## Recommended Order

1. [The DevOps Handbook](The%20DevOps%20Handbook-%5BGene%20Kim%5D.pdf)
2. [Release It! Design and Deploy Production-Ready Software](Release%20It!%20Design%20and%20Deploy%20Production-Ready%20Software.pdf)
3. [Observability Engineering: Achieving Production](observability-engineering-achieving-production.pdf)
4. [Linux Utilities Cookbook](Linux%20Utilities%20Cookbook-%5BJames%5D.pdf)

## Book Notes

- [The DevOps Handbook](The%20DevOps%20Handbook-%5BGene%20Kim%5D.pdf): Delivery pipelines, culture, and operational flow.
- [Release It! Design and Deploy Production-Ready Software](Release%20It!%20Design%20and%20Deploy%20Production-Ready%20Software.pdf): Production stability patterns and resilience.
- [Observability Engineering: Achieving Production](observability-engineering-achieving-production.pdf): Metrics, traces, and logs with production intent.
- [Linux Utilities Cookbook](Linux%20Utilities%20Cookbook-%5BJames%5D.pdf): Practical command-line tooling for operations work.
